# Installation

MessageKit is a powerful framework that simplifies the process of building messaging bots on [xmtp](https://docs.xmtp.org). Hereâ€™s a step-by-step guide to get you started.

::::steps

### CLI

This tool simplifies the creation and configuration of new apps.

:::code-group

```bash [bun]
bun create message-kit
```

```bash [yarn]
yarn create message-kit
```

```bash [npx]
npx create-message-kit@latest
```

:::

_Refer to [structure](/concepts/structure) of the app so see the folder is created._

### Example

Start with a simple app that replies with `gm`.

```tsx [src/index.ts]
run(async (context: HandlerContext) => {
  // Get the message and the address from the sender
  const { content, sender } = context.message;

  // To reply, just call `reply` on the HandlerContext.
  await context.send(`gm`);
});
```

_You can also pick one of the 2 full [templates](#templates) available._

### Variables

Create a `.env` file in the root of your project

```bash [cmd]
KEY= # the private key of the bot wallet
```

### Run the bot

Follow the steps below to run the bot

:::code-group

```bash [yarn]
# Install the dependencies
yarn install
# Run the app
yarn dev
```

```bash [bun]
# Install the dependencies
bun install
# run the app
bun dev
```

```bash [npm]
# Install the dependencies
npm install
# Run the app
npm run dev
```

:::

### Test

Send a message to the app, and it will reply `gm`!

## Templates

- [`gm`](./templates/gm): Simple gm bot example.
- [`group`](./templates/group): Group chat bot example with commands and examples.
- [`ens-agent`](./templates/ens-agent): ENS agent for registering and checking domains

## Manual Installation

You can also install the package manually using your preferred package manager.

:::code-group

```bash [bun]
bun i @xmtp/message-kit
```

```bash [npm]
npm i @xmtp/message-kit
```

```bash [yarn]
yarn add @xmtp/message-kit
```

:::
