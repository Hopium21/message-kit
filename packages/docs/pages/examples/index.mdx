# Examples

Here are some ready to go examples of mini-apps built with MessageKit. Documented and ready to use.

## Examples

- [`gm`](./gm): Simple gm bot example.
- [`group`](./group): Group chat bot example with commands and examples.
- [`ens-agent`](./ens-agent): ENS agent for registering and checking domains

And here are some simpler quickstarts for different use cases.

### Quickstarts

- [Event invite bot](https://github.com/fabriguespe/invite-bot): Invite bot for Events linked to a Notion db.
- [Simple AI bot](https://github.com/fabriguespe/gpt-example): Simple GPT example.
- [Subscription bot](https://github.com/fabriguespe/subscription-bot): Subscription bot example for a newsletter.
- [Next js](https://github.com/fabriguespe/message-kit-next-quickstart): Deploy in a vercel app.
- [Gated group](https://github.com/ephemerahq/gated-group): Group chat with a bot that can add members to a group.

### Frames quickstarts

- [Base Tx Frame](../directory/basetxframe.mdx): A dev-oriented frame designed for creating transactions on Base.
- [OnchainKit Quickstart](https://github.com/ephemeraHQ/open-frames-starter-onchainkit/): Onchain Kit quickstart that integrates XMTP.
- [Frog Quickstart](https://github.com/ephemeraHQ/open-frames-starter-frog): Frog XMTP quickstart
- [Frames.js Quickstart](https://github.com/ephemeraHQ/open-frames-starter-framesjs/): Frames.js quickstart that integrates XMTP.
