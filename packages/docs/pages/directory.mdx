# App directory

Discover some of the most popular Apps built with MessageKit.

## Apps

- 🚀 [`trendingmints.eth`](https://github.com/fabriguespe/trendingmints-bot) : Subscribe to get real-time trending mints through [Airstack](https://www.airstack.xyz/)
- 💧 [`faucetbot.eth`](https://github.com/fabriguespe/faucet-bot) : Delivers Faucet funds to devs on Testnet via [Learnweb3](https://learnweb3.io/faucets/).
- 🛒 [`thegeneralstore.eth`](https://github.com/fabriguespe/thegeneralstore-bot) : E-commerce app for ordering goodies in hackathons.

## Frames

All these Frames are compatible with the [Open Frames](https://www.openframes.xyz/) standard.

- 🧠 [Brain Frame](https://frame.brianknows.org/): AI multipurpose frame.
- 💸 [Tx Receipt](https://github.com/fabriguespe/tx-receipt-frame) : Frame that acts a transaction receipt.
- 💸 [BaseFrame](./directory/baseframe.mdx): A dev-oriented frame designed for creating transactions on Base.
- 📊 [Polls](https://fc-polls-five.vercel.app/): Create polls on messaging apps.
- 🌿 [Mint Frame](https://github.com/fabriguespe/mint-frame/): Frame for redirecting users to mint on Zora

## Games

- 📅 [`dailywordle.eth`](https://converse.xyz/dm/dailywordle.eth) : Play daily to the WORDLE game by [du8](https://warpcast.com/ds8/0x2d31015a) through messaging.
- 🖼️ [Wordle](https://openframedl.vercel.app/): Wordle in a frame - daily or random game.
- 🌍 [Farguessr](https://farguessr.vercel.app/): Guess the distance between 2 countries
- 🪨 [Rock Paper Scissors](https://xmtp-frame-rock-paper-scissors.vercel.app/): Rock paper scissors game.
- 🎰 [Slot machine](https://slot-machine-frame.vercel.app/): Slot machine open frame.
