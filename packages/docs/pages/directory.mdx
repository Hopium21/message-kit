# Directory

These are live bots and open frames built with MessageKit and Open Frames.

## Agents

#### 💳 [Base Agent](https://converse.xyz/dm/baseagent.converse.xyz)

- **Description**: Base agent for all your onchain tasks by prompting in natural language
- **Learn more**: [Landing page](./directory/baseagent)

#### 🔗 [ENS Agent](https://converse.xyz/dm/ensagent.converse.xyz)

- **Description**: ENS agent for registering and checking domains.
- **Learn more**: [Landing page](./directory/ensagent)

#### 🧠 [Brian Bot](https://converse.xyz/dm/brian-agent.eth)

- **Description**: Brian Team's agent for all your onchain tasks by prompting in natural language
- **Learn more**: [Landing page](https://www.brianknows.org/)

#### 📅 [Wordle Bot](https://converse.xyz/dm/wordlebot.eth)

- **Description**: Play daily Framdle in group chats.
- **Source Code**: [Bot](https://github.com/ephemerahq/wordle-bot)

## Open Frames

- 🧠 [Brain Frame](https://frame.brianknows.org/): AI multipurpose frame.
- 🖼️ [Framedl](https://framedl.xyz/): Wordle in a frame - daily or random game.
- 🪨 [ENS Domain Tool](https://ens.steer.fun/): ENS bot for messaging apps. [Repo](https://github.com/stephancill/ens-frame-v2)
- 🎰 [Slot machine](https://slot-machine-frame.vercel.app/): Slot machine open frame.
