# Directory

These are live bots and open frames built with MessageKit and Open Frames.

## Agents

#### 🧠 [Brian Bot](https://converse.xyz/dm/brian-agent.eth)

- **Description**: Brian Team's agent for all your onchain tasks by prompting in natural language

#### [Base Agent](https://converse.xyz/dm/baseagent.converse.xyz)

- **Description**: Base agent for all your onchain tasks by prompting in natural language
- **Source Code**: [Bot](https://github.com/fabriguespe/baseagent)

#### 🕸️ [ENS agent](https://converse.xyz/dm/ensagent.converse.xyz)

- **Description**: ENS agent for registering and checking domains.
- **Source Code**: [Bot](https://github.com/fabriguespe/ens-agent) | [Frame](https://github.com/stephancill/ens-frame-v2)

#### 📅 [Wordle Bot](https://converse.xyz/dm/wordlebot.eth)

- **Description**: Play daily Framdle in group chats.
- **Source Code**: [Bot](https://github.com/ephemerahq/wordle-bot)

#### 💧 [Faucetbot](https://converse.xyz/dm/faucetbot.eth)

- **Description**: Delivers faucet funds to devs on testnet.
- **Source Code**: [Bot](https://github.com/fabriguespe/faucet-bot) | [Frame](https://github.com/fabriguespe/tx-receipt-frame)

#### 🛒 [The general store](https://converse.xyz/dm/thegeneralstore.eth)

- **Description**: E-commerce app for ordering goodies in hackathons using a live notion doc as a prompt.
- **Source Code**: [Bot](https://github.com/fabriguespe/thegeneralstore-bot) | [Prompt](https://ephemerahq.notion.site/General-Store-Prompt-11830823ce9280699fdef0655dce7315?pvs=4)

#### 🪨 [URL to mint frame](https://converse.xyz/dm/urltomint.eth)

- **Description**: Turn a Zora or Coinbse Wallet NFT url into a Mint Open Frame that works in messaging apps.
- **Source Code**: [Bot](https://github.com/fabriguespe/url-to-cb-mint) | [Frame](https://github.com/builders-garden/xmtp-coinbase-mint-frame)

## Open Frames

- 🧠 [Brain Frame](https://frame.brianknows.org/): AI multipurpose frame.
- 🖼️ [Framedl](https://framedl.xyz/): Wordle in a frame - daily or random game.
- 🪨 [ENS Domain Tool](https://ens.steer.fun/): ENS bot for messaging apps. [Repo](https://github.com/stephancill/ens-frame-v2)
- 🎰 [Slot machine](https://slot-machine-frame.vercel.app/): Slot machine open frame.
