# Middlewares

Middlewares are functions that can be used to modify the behavior of a message. Just copy the middleware you want to use and paste it in your code.

## Middleware list

- [OpenAI](/middlewares/gpt): Use OpenAI to generate responses.
- [Cron](/middlewares/cron): Schedule messages to be sent at a specific time.
- [Redis](/middlewares/redis): Use Redis as a database.
- [Stackso](/middlewares/stackso): Use Stackso to earning points in web3.
- [Notion](/middlewares/notion): Use Notion as a database.
- [Resolver](/middlewares/resolver): Use Resolver to resolve addresses.
