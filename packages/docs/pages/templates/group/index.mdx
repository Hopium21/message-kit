# Group chat

This is a reference app that will allow you to understand the potential and all of its features. Through many handlers, you will be able to understand different use cases:

## Handlers

- [Tipping](/templates/group/handlers/tipping): Tip when a user replies with `/tip`.
- [Games](/templates/group/handlers/games): Handle frame games like `/game slot`, `/game wordle`
- [Transactions](/templates/group/handlers/transactions): Handle skills like `/send`, `/swap`, `/show`.
- [Helpers](/templates/group/handlers/helpers): Handle group helpers like `/id`, `/help`

## Main code

```tsx [src/index.ts]
v;
import { run, HandlerContext } from "@xmtp/message-kit";

// Main function to run the app
run(async (context: HandlerContext) => {
  const {
    message: { typeId },
  } = context;

  if (!context.group) {
    context.send(
      "This agent only works in group chats. Please add this agent to a group to continue",
    );
    return;
  }
});
```

## Run the app

Follow the steps below to run the app

::::steps

### Setup

```bash [cmd]
# Clone the repo
git clone https://github.com/ephemeraHQ/message-kit
# Go to the templates/group folder
cd templates/group
# Install the dependencies
yarn install
# Run the app
yarn dev
```

### Variables

Set up these variables in your app

```bash [cmd]
KEY= # the private key of the agent wallet
OPEN_AI_API_KEY= # your openai api key
```
