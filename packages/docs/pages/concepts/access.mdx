# Access

## Grant access

Create the `accessHandler` function to handle access to a group chat. Returns `true` or `false` based on the access request.

```tsx [src/index.ts]
type AccessHandler = (request: any) => Promise<boolean>;

const accessHandler: AccessHandler = async (request) => {
  const { sender, conversationId } = request;
  if (sender) return true;
  return false;
};
```

## Send it as a parameter

To declare the access command in the app, you need to add the following code:

```tsx [src/index.ts]
const appConfig = {
  accessHandler: accessHandler,
};

run(async (context: HandlerContext) => {
  /* app logic here */
}, appConfig);
```

## Full example

```jsx [src/index.ts]
import "dotenv/config";
import { run, HandlerContext } from "@xmtp/message-kit";

type AccessHandler = (request: any) => Promise<boolean>;

const accessHandler: AccessHandler = async (request) => {
  const { sender, conversationId } = request;
  if (sender) return true;
  return false;
};

const appConfig = {
  accessHandler: accessHandler,
};

// Entry point for running the app
run(async (context: HandlerContext) => {
  context.reply("Welcome");
}, appConfig);
```
