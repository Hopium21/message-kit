# Groups

Groups in XMTP enable secure and private communication among multiple members. For more details, see XMTP's [Groups documentation](https://xmtp.org/docs/build/group-chat).

## Create a new group

Using the client you can create a new group with an array of addresses.

```tsx
// get the client
const { client } = context;
// create a new group
const group = await client.conversations.newGroup(addresses);
```

## Sync a group

Each time you perform an action on a group you must sync it to get the latest state.

```tsx
// get the conversation
const { conversation } = context;
// sync the conversation
await conversation.sync();
```

## Group metadata

> You need admin privileges to update the metadata.

Groups have the following metadata:

### Group name

You can update the name of the group chat.

```tsx
// get the conversation
const { conversation } = context;
// get the name
const groupName = conversation.name;
// update the name
await conversation.updateName(name);
```

### Group description

You can update the description of the group chat.

```tsx
// get the conversation
const { conversation } = context;
// get the description
const groupDescription = conversation.description;
// update the description
await conversation.updateDescription(description);
```

### Group image

You can update the image of the group chat.

```tsx
// get the conversation
const { conversation } = context;
// get the image
const groupImage = conversation.imageUrl;
// update the image
await conversation.updateImageUrl(imageUrl);
```

## Member management

> You need admin privileges to manage members.

You can add or remove members from a group.

### Remove members from a group

```tsx
// get the conversation
const { conversation } = context;
//By address
await conversation.removeMembers(userAddresses);
//By inboxId
await conversation.removeMembersByInboxId(removedInboxes);
```

### Add members to a group

```tsx
// get the conversation
const { conversation } = context;
//By address
await conversation.addMembers(userAddresses);
//By inboxId
await conversation.addMembersByInboxId(addedInboxes);
```

## Messages

To get the message history in a group you can use the `messages` property.

```tsx
// get the conversation
const { conversation } = context;
// sync group first
await conversation.sync();
// get messages
const messages = conversation.messages();
```
