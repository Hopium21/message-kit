# Examples

> 💬 **Try it:** Message `awesome.hi.xmtp.eth`

### Bots 🤖

- 🚀 [`trendingmints.eth`](https://github.com/fabriguespe/trendingmints-bot) : Subscribe to get real-time trending mints in Base through [Airstack](https://www.airstack.xyz/)
- 💧 [`faucetbot.eth`](https://github.com/fabriguespe/faucet-bot) : Delivers Faucet funds to devs on Testnet via [Learnweb3](https://learnweb3.io/faucets/).
- 🤖 [`docs.hi.xmtp.eth`](https://github.com/fabriguespe/xmtp-docs-bot) : Chat with the XMTP [Docs](https://xmtp.org/docs/introduction) through [Kapa](https://kapa.ai).
- 🛒 [`thegeneralstore.eth`](https://github.com/fabriguespe/thegeneralstore-bot) : E-commerce bot for ordering goodies in hackathons.
- 📅 [`dailywordle.eth`](https://converse.xyz/dm/dailywordle.eth) : Play daily to the WORDLE game by [du8](https://warpcast.com/ds8/0x2d31015a) through messaging.
- 🃏 [`gpt.hi.xmtp.eth`](https://github.com/fabriguespe/crypto-joker) : Get funny cypto jokes with an a GPT powered bot using Open AI APIs.
