# Admin

This example shows how to create an admin bot that will allow you to block and unblock users.

## Commands

Here are the commands to block and unblock users:

```bash [cmd]
/block @alice
/unblock @alice
```

::::note[Commands]
_See how commands are defined in [`src/commands.ts`](https://github.com/xmtp-labs/botkit/blob/main/examples/group/src/commands.ts)_
